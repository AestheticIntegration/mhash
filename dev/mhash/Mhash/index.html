type 'a hasher = {
hash_into : ctx out. ('ctx, 'out) hash_algo -> 'ctx -> 'a -> unit;
}

A hash function for a type 'a.

It updates the given ctx using the HASH_ALGO core functions and other hashable instances. This works for any concrete hash function that implements HASH_ALGO.
